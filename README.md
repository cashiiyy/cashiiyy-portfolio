# Student Portfolio - Kasinathan P S

A minimalist, professional portfolio website for an AIML Engineering student, focusing on the intersection of Hardware and Intelligence.

## Features

- **Clean Aesthetic**: Strict Black & White design.
- **Responsive**: Fully responsive flexbox/grid layout.
- **Tech Stack**: HTML5, CSS3, Vanilla JavaScript.

## Sections

- **The Engineer**: Philosophy and Bio.
- **Capabilities**: Skills in Electronics, AI, and Tools.
- **Featured Work**: Highlighting the "Vidya" AI assistant project.
- **Connect**: Social links and contact info.

## Deployment

This project is ready for **GitHub Pages**.

1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select `main` branch and `/ (root)` folder.
4. Save.
